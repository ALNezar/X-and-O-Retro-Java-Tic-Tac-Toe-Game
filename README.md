## 1. **JUST CLICK THE `Tic-tac-toe.JAR`!**

**OR**

## 2. **JUST CLICK THE `Tic-tac-toe.EXE`!**

---

## 3. **Download or clone this project:**

```sh
git clone https://github.com/ALNezar/X-and-O.git
cd X-and-O
```

---

## 4. **Run the game:**

**‚ñ∏ If you use an IDE (like IntelliJ IDEA or Eclipse):**
Open the project and run `Main.java` directly. The IDE will automatically handle the necessary dependencies.

**‚ñ∏ If you use the command line:**

**For Windows:**

```sh
javac -cp ".;sqlite-jdbc-3.27.2.1.jar" *.java
java -cp ".;sqlite-jdbc-3.27.2.1.jar" Main
```

**For macOS/Linux:**

```sh
javac -cp ".:sqlite-jdbc-3.27.2.1.jar" *.java
java -cp ".:sqlite-jdbc-3.27.2.1.jar" Main
```

# FILES IN THIS PROJECT

* `Main.java` ‚Äì run de game!
* `TicTacToeApp.java` ‚Äì Manages the overall app and screens
* `GamePanel.java` ‚Äì The main game screen
* `WelcomePanel.java` ‚Äì The welcome and main menu
* `SettingsPanel.java` ‚Äì Handles game settings
* `Settings.java` ‚Äì Stores game settings in memory
* `resources/` ‚Äì Contains game assets like audio files (`.wav`)
* `sqlite-jdbc-3.27.2.1.jar` ‚Äì The JDBC driver for SQLite database operations
* `settings.db` ‚Äì The SQLite database file (for persistent settings)
